# Журнал известных проблем

Назначение файла:
- фиксировать реальные ошибки, найденные в валидаторе и процессе перевода;
- хранить решение и ссылку на regression test.

## Правила ведения

- Каждую новую проблему оформлять отдельной записью с уникальным ID.
- Для каждой записи обязательно указывать статус: open или resolved.
- Для каждого исправления обязательно указывать имя regression test.
- Изменения валидатора без соответствующего теста не считаются завершенными.
